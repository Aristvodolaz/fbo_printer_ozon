# Быстрый старт

## Для Windows

1. **Установите Node.js** (если еще не установлен)
   - Скачайте с https://nodejs.org/
   - Выберите LTS версию

2. **Установите зависимости**
   - Запустите `install.bat` (двойной клик)
   - Или в командной строке: `npm install`

3. **Запустите сервер**
   - Запустите `start.bat` (двойной клик)
   - Или в командной строке: `npm start`

4. **Откройте в браузере**
   - `http://localhost:3000`

## Для Linux/Mac

1. **Установите Node.js** (если еще не установлен)
   ```bash
   # Ubuntu/Debian
   sudo apt-get install nodejs npm
   
   # Mac (через Homebrew)
   brew install node
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Запустите сервер**
   ```bash
   npm start
   ```

4. **Откройте в браузере**
   - `http://localhost:22`

**⚠️ Внимание:** Порт 22 обычно используется для SSH. Если возникнет конфликт, используйте другой порт: `PORT=3022 npm start`

## Для запуска на сервере

См. подробную инструкцию в `DEPLOY.md`

**Важно:** 
- Сервер автоматически раздает `index.html` как главную страницу
- API доступен по адресу `/api/*`
- Прогресс сохраняется в БД автоматически
